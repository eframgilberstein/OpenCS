# OpenCS

A lightweight, modular PHP web application for managing users, bookings, and content. OpenCS provides a simple foundation for building booking systems or content-driven platforms using core PHP and MySQL without external frameworks.

---

## Overview

OpenCS is designed as a clean and extensible starting point for developers who want full control over their application stack. The project demonstrates authentication workflows, booking management, session handling, and basic administrative functionality using structured PHP files.

This project is ideal for:

- Learning backend web development with PHP
- Building a simple booking system
- Creating a custom content management workflow
- Extending into a full MVC application

---

## Features

- User registration and authentication
- Secure login and logout handling
- Admin dashboard
- Booking creation and management
- Booking overview page
- Basket/cart-style interaction
- Modular PHP file structure
- Custom CSS styling
- MySQL database integration
